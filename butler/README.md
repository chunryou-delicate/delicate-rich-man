## II. 가변 적립 배분 엔진 (allocate.py) ✅
입력(이번 금액 + 자산별 잔고) → policy 비율로 배분 지시. **알고리즘(결정론):**
1. 후잔고 목표액 = (현잔고+입력) × 목표비중. 자산별 미달분(deficit) 계산.
2. 입력액 ≤ 총미달: 미달에 **비례 배분**(목표미달 자산 우선, 초과 자산 0).
3. 입력액 > 총미달: 미달 다 채우고 나머지는 목표비중대로.
4. 계좌: 현금→유동성버퍼(CMA), ETF→연금(한도)→ISA(한도)→일반.
5. 리밸: 후비중이 목표 ±5%p 초과 자산 플래그(다음 적립으로 완화, 밴드 큰 초과만 부분매도).

검산(예시 300만, 잔고 국내400·글로벌200·채권300·현금100만): 글로벌 최대미달 → +225만,
국내 +48.5만, 채권 초과 0, 현금 +26.5만. 계좌: 현금 CMA, ETF 273.5만 → 연금150·ISA123.5만. ✓
